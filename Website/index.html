Introduction

Internet voting, or i-Voting, is a system that allows voters to cast their ballots from any internet-connected computer anywhere in the world. Completely unrelated to the electronic voting systems used elsewhere, which involve costly and problematic machinery, the solution is simple, elegant and secure.

Our Project

Our project objective will make an efficient way to conducting any type of Election . Our stakeholder will derived from organization to government . One of the feature that our system can have is the voters will get Unique ID and Password, Using which they can vote for a Candidate only once per Election . Admin of the election can see identification of each voters from the database and also the admin can set the option to delete particular vote of the voters
